## 💡 Key Learnings

* Linear search, prefix logic, and brute-force foundations
* Dutch National Flag algorithm (3-way partitioning)
* Moore's Voting Algorithm (majority element)
* Two-pointer technique (merge sorted, 3Sum, 4Sum)
* Sliding window (min subarray sum, longest subarray)
* Prefix sum + HashMap (subarray sum = K, XOR = K)
* Next permutation (in-place lexicographic ordering)
* In-place matrix operations (rotate 90°, set zeroes, spiral)
* Interval merging logic (greedy sorting + sweep)
* Kadane's algorithm extension (max product subarray)
* Bit manipulation (single number, XOR tricks)
* Modified merge sort (reverse pairs — inversion count)
* Pascal's Triangle construction (combinatorics pattern)
* Binary exponentiation (fast power in O(log n))
* Hash-based O(n) solutions vs. sorting-based O(n log n) tradeoffs
